# Half Adder Circuit in VHDL

This repository contains a clean, professional, and well-commented VHDL implementation of a digital **Half Adder** circuit, designed as part of the *Technische Informatik* course (SoSe 2026).

## Circuit Specifications
A Half Adder is a combinational circuit that performs the addition of two 1-bit binary numbers. It produces two outputs: **Sum** and **Carry**.

- **Inputs:** `a`, `b` (1-bit)
- **Outputs:** `sum` ($a \oplus b$), `carry` ($a \cdot b$)

## VHDL Implementation
The implementation uses standard IEEE library types and follows best practices for clean hardware description style.
